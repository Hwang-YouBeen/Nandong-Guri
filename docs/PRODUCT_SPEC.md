# 난동구리 제품 기능 명세

## 1. 제품 정보

- 한글명: **난동구리**
- 영문명: **Nandong Guri**
- 지원 운영체제: **macOS, Windows**
- 첫 MVP 지원 서비스: **ChatGPT 웹, Gemini 웹**
- 첫 MVP 지원 브라우저: **Chrome, Microsoft Edge**

## 2. 한 줄 설명

ChatGPT 또는 Gemini의 답변이 완료되면 작은 3D 너구리 한 마리가 화면에 즉시 나타나고, 너구리를 클릭하면 완료된 답변으로 이동하는 데스크톱 알림 앱.

## 3. 제품 목표

사용자가 생성형 AI의 답변을 기다리는 동안 다른 작업에 집중할 수 있도록 한다. 평범한 알림창 대신 화면을 돌아다니고 춤추는 너구리를 사용해 답변 완료를 확실하고 재미있게 알린다.

## 4. 확정 원칙

- 너구리는 항상 한 마리다.
- 답변이 완료되면 즉시 등장한다.
- 사용자는 난동 강도를 선택할 수 없다.
- 사용자가 반응하지 않으면 5초 간격으로 난동 단계가 상승한다.
- 효과음은 기본으로 켜져 있다.
- 효과음은 **최초 등장할 때**와 **사용자가 너구리를 클릭할 때**만 재생한다.
- 회의, 화면 공유 또는 전체 화면 상태에서도 자동으로 억제하지 않는다.
- 여러 모니터가 연결되어 있으면 너구리가 모니터 사이를 이동한다.
- 사용자는 너구리 이외의 화면 영역을 평소처럼 클릭할 수 있어야 한다.
- 너구리는 화면 경계 안에서만 움직인다.

## 5. 핵심 사용자 흐름

1. 사용자가 난동구리 데스크톱 앱을 실행한다.
2. 앱은 macOS 메뉴바 또는 Windows 시스템 트레이에 상주한다.
3. 사용자가 ChatGPT 또는 Gemini에 질문한다.
4. 사용자는 답변 생성 중 다른 작업을 한다.
5. 브라우저 확장 프로그램이 답변 완료를 감지한다.
6. 시스템 알림과 최초 등장 효과음이 발생한다.
7. 작은 3D 너구리가 현재 화면에 즉시 나타난다.
8. 너구리는 짧게 걷거나 뛴 뒤 한자리에서 춤을 춘다.
9. 너구리는 다시 짧게 이동하고 다른 동작이나 춤을 반복한다.
10. 사용자가 너구리를 클릭하면 클릭 효과음이 재생되고 완료된 답변 페이지로 이동한다.
11. 답변 페이지가 열리면 너구리가 사라진다.

사용자가 답변 완료 순간 이미 해당 ChatGPT 또는 Gemini 탭을 보고 있다면 너구리는 약 5초 동안 등장한 뒤 자동으로 사라진다.

## 6. 너구리 행동

### 기본 이동 패턴

- 화면 전체의 위쪽과 아래쪽을 모두 사용한다.
- 화면 밖으로 몸이 잘리지 않도록 모델 크기를 고려해 이동 범위를 제한한다.
- 계속 도망치듯 움직이지 않는다.
- **짧게 이동 → 제자리에서 4~7초 춤 → 다시 짧게 이동** 패턴을 반복한다.
- 걷기, 달리기, 몰래 걷기 모션과 실제 화면 이동 방향을 맞춘다.
- 이동 방향이 바뀌면 몸의 방향도 자연스럽게 바뀐다.

### 동작 종류

- 걷기와 달리기
- 꼬리 흔들기
- 힙합과 삼바 춤
- 문워크와 브레이크댄스
- 복싱과 발차기
- 화면 상단 또는 창 상단 바에 매달리는 연출
- 여러 모니터 사이 이동

### 난동 단계

- 답변 완료 직후 0단계로 시작한다.
- 사용자가 클릭하지 않으면 5초마다 최대 3단계까지 상승한다.
- 단계가 높아질수록 더 과격하고 다양한 애니메이션을 선택한다.
- 단계가 높아져도 너구리는 계속 클릭할 수 있어야 한다.

## 7. 클릭과 화면 조작

- 투명 오버레이 전체가 다른 앱의 클릭을 막아서는 안 된다.
- 마우스가 너구리의 현재 영역에 있을 때만 오버레이가 클릭을 받는다.
- 너구리 영역에서는 포인터 커서를 표시한다.
- 너구리의 클릭 영역에는 작은 여유 공간을 두어 움직이는 중에도 클릭하기 쉽게 한다.
- 너구리를 클릭하면 새 브라우저 창을 만들지 않고, 답변을 생성했던 기존 ChatGPT 또는 Gemini 탭과 브라우저 창을 앞으로 가져온다.
- 긴급 퇴장 단축키는 `Command/Ctrl + Shift + G`이다.

## 8. 종료 조건

- 사용자가 너구리를 클릭한다.
- 사용자가 완료된 답변 탭을 직접 확인한다.
- 이미 답변 탭을 보고 있던 상태에서 약 5초가 지난다.
- 사용자가 긴급 퇴장 단축키를 누른다.

## 9. 메뉴바와 시스템 트레이

### macOS

- 앱을 실행하면 Dock 중심의 일반 앱이 아니라 상단 메뉴바에 상주한다.
- 메뉴바 아이콘을 왼쪽 클릭하면 작은 설정 패널을 열거나 닫는다.
- 메뉴바 아이콘을 오른쪽 클릭하면 `설정 열기`, `난동구리 종료` 빠른 메뉴를 표시한다.
- 설정창의 닫기 동작은 앱 종료가 아니라 창 숨김이다.

### Windows

- 앱을 실행하면 작업 표시줄 오른쪽 알림 영역의 시스템 트레이에 상주한다.
- 아이콘이 보이지 않으면 숨겨진 아이콘 `^` 목록에서 찾을 수 있다.
- 시스템 트레이 아이콘을 왼쪽 클릭하면 설정 패널을 열거나 닫는다.
- 오른쪽 클릭하면 빠른 메뉴를 표시한다.
- 설정창을 닫아도 백그라운드 감지는 계속된다.

## 10. 설정 패널

설정 패널은 큰 웹 대시보드가 아니라 RunCat과 비슷한 작은 로컬 앱 패널을 사용한다. 너구리 털의 검정, 짙은 회색, 중간 회색, 크림색을 주 색상으로 사용한다.

### 왼쪽: 기능 조절

- 효과음 켜기 및 끄기
- 시스템 알림 켜기 및 끄기
- 로그인 시 자동 실행 켜기 및 끄기
- 너구리 크기 조절
- 긴급 퇴장 단축키 확인

### 오른쪽: 정보 메뉴

- 난동구리 소개
- 사용 방법
- 확장 프로그램 설치
- 환경 및 단축키

각 항목을 클릭하면 현재 패널 위에 별도의 상세 설명 시트가 열린다.

설정 패널에는 여러 포즈의 2D 너구리를 장식으로 배치한다. 장식 이미지는 제목, 설정명, 스위치와 설명을 가리지 않아야 하며 최종 위치는 `desktop-app/src/App.css`의 mascot 클래스에서 관리한다.

설정창 오른쪽 위 `×` 버튼은 앱 종료가 아니라 설정창 숨김으로 동작한다.

## 11. 브라우저 확장 프로그램

- Manifest V3 확장 프로그램을 사용한다.
- ChatGPT와 Gemini의 답변 완료 상태를 감지한다.
- Gemini는 중지 버튼 상태와 마지막 답변 텍스트의 변화가 멈추는 상태를 함께 사용해 완료를 판정한다.
- 동일한 답변 카드에서는 완료 이벤트를 한 번만 전달한다.
- 완료된 답변의 서비스명과 URL을 데스크톱 앱으로 전달한다.
- 데스크톱 앱과 `http://127.0.0.1:43119` 로컬 HTTP 브리지로 통신한다.
- 사용자가 최초 한 번 수동으로 설치해야 한다.
- 설치 후 ChatGPT와 Gemini의 기존 탭을 반드시 새로고침해야 한다.
- 자세한 설치 방법은 [확장 프로그램 설치 가이드](EXTENSION_INSTALL.md)를 따른다.

## 12. 연결 상태

- `로컬 브리지 연결`: 데스크톱 앱 내부 서버가 정상 작동하는 상태
- `브라우저 자동 감지 연결`: 확장 프로그램이 최근 15초 이내에 앱으로 신호를 보낸 상태
- 두 연결이 모두 정상이면 설정 패널에 `준비됨`을 표시한다.

## 13. 기술 구성

- 데스크톱 프레임워크: Tauri 2
- 네이티브 로직: Rust
- 설정 UI: React, TypeScript, Vite
- 3D 렌더링: Three.js
- 3D 파일: 텍스처와 애니메이션을 포함한 GLB
- 브라우저 확장: Manifest V3
- 로컬 통신: `127.0.0.1:43119` HTTP 브리지

## 14. 현재 MVP 범위 밖

- Safari와 Firefox 지원
- ChatGPT·Gemini 이외의 생성형 AI 서비스
- 여러 마리의 너구리
- 사용자가 직접 고르는 난동 강도
- 모바일 앱
- 앱스토어 및 브라우저 스토어 자동 배포

데스크톱 앱, 브라우저 확장 프로그램, 제품 문서와 3D 제작 원본은 서로 분리해 관리한다. 자세한 경로와 각 폴더의 역할은 [프로젝트 구조 문서](PROJECT_STRUCTURE.md)를 따른다.
