# 난동구리 확장 프로그램 설치 가이드

이 문서는 브라우저 확장 프로그램을 한 번도 설치해 본 적 없는 사람도 따라 할 수 있도록 작성했습니다.

난동구리를 사용하려면 다음 두 가지가 모두 필요합니다.

1. **난동구리 데스크톱 앱**: 화면에 3D 너구리를 보여줍니다.
2. **난동구리 브라우저 확장 프로그램**: ChatGPT와 Gemini의 답변이 끝났는지 확인합니다.

확장 프로그램은 현재 Chrome과 Microsoft Edge를 지원합니다. Safari와 Firefox는 아직 지원하지 않습니다.

## 설치하기 전에 확인할 것

1. 난동구리 프로젝트 폴더를 Finder 또는 파일 탐색기에서 찾습니다.
2. 그 안에 `browser-extension`이라는 폴더가 있는지 확인합니다.
3. `browser-extension` 폴더 안에는 다음 파일이 보여야 합니다.
   - `manifest.json`
   - `content.js`
   - `icons` 폴더
4. 설치할 때는 파일 하나가 아니라 **`browser-extension` 폴더 자체**를 선택합니다.

폴더 위치는 다음과 같습니다.

```text
nandong_guri
└── browser-extension  ← 이 폴더를 선택합니다
    ├── manifest.json
    ├── content.js
    └── icons
```

## macOS에서 Chrome에 설치하기

1. 먼저 난동구리 데스크톱 앱을 실행합니다.
2. 화면 위쪽 메뉴바에 난동구리 아이콘이 나타나는지 확인합니다.
3. Chrome을 엽니다.
4. Chrome 화면 맨 위의 긴 주소 입력 칸을 한 번 클릭합니다.
5. 주소창에 `chrome://extensions`를 정확히 입력하고 `Return` 키를 누릅니다.
   - Google 검색창이 아니라, 웹사이트 주소가 표시되는 **맨 위 주소창**에 입력해야 합니다.
6. `확장 프로그램` 화면이 열리면 오른쪽 위의 **개발자 모드** 스위치를 찾습니다.
7. 개발자 모드 스위치를 눌러 켭니다. 스위치 색이 바뀌면 켜진 것입니다.
8. 화면 위쪽에 나타난 **압축해제된 확장 프로그램을 로드합니다** 버튼을 누릅니다.
   - Chrome 버전에 따라 `압축해제된 확장 프로그램을 로드`라고 표시될 수도 있습니다.
9. 폴더를 고르는 창이 열리면 `nandong_guri` 프로젝트 폴더로 이동합니다.
10. 프로젝트 안의 `browser-extension` 폴더를 한 번 클릭합니다.
11. 오른쪽 아래의 **선택** 또는 **열기** 버튼을 누릅니다.
12. 확장 프로그램 목록에 **난동구리 브라우저 감지기** 카드가 나타나는지 확인합니다.
13. 카드 오른쪽 아래의 스위치가 켜져 있는지 확인합니다.
14. 이미 열려 있던 ChatGPT와 Gemini 탭으로 이동해 각각 새로고침합니다.
   - 새로고침 단축키는 `Command + R`입니다.
15. 화면 위쪽 메뉴바의 난동구리 아이콘을 왼쪽 클릭합니다.
16. 설정 패널 위쪽 상태가 **준비됨**으로 바뀌면 설치가 끝난 것입니다.

확장 프로그램 파일이 업데이트된 경우 `chrome://extensions`에서 **난동구리 브라우저 감지기** 카드의 새로고침 아이콘을 누른 뒤 ChatGPT와 Gemini 탭도 다시 새로고침합니다.

## Windows에서 Chrome에 설치하기

1. 먼저 난동구리 데스크톱 앱을 실행합니다.
2. 작업 표시줄 오른쪽 아래 알림 영역에 난동구리 아이콘이 있는지 확인합니다.
   - 보이지 않으면 `^` 모양의 숨겨진 아이콘 표시 버튼을 누릅니다.
3. Chrome을 엽니다.
4. 화면 맨 위 주소창에 `chrome://extensions`를 입력하고 `Enter` 키를 누릅니다.
5. 오른쪽 위의 **개발자 모드** 스위치를 켭니다.
6. 화면 위쪽의 **압축해제된 확장 프로그램을 로드합니다** 버튼을 누릅니다.
7. 폴더 선택 창에서 난동구리 프로젝트 안의 `browser-extension` 폴더를 선택합니다.
8. **폴더 선택** 버튼을 누릅니다.
9. **난동구리 브라우저 감지기** 카드가 나타나고 스위치가 켜져 있는지 확인합니다.
10. ChatGPT와 Gemini 탭을 각각 새로고침합니다.
    - 새로고침 단축키는 `Ctrl + R`입니다.
11. 작업 표시줄의 난동구리 아이콘을 왼쪽 클릭합니다.
12. 설정 패널 상태가 **준비됨**으로 바뀌면 설치가 끝난 것입니다.

## Windows에서 Microsoft Edge에 설치하기

1. 난동구리 데스크톱 앱을 실행합니다.
2. Microsoft Edge를 엽니다.
3. 화면 맨 위 주소창에 `edge://extensions`를 입력하고 `Enter` 키를 누릅니다.
4. 왼쪽 아래 또는 화면 한쪽에 있는 **개발자 모드** 스위치를 켭니다.
5. **압축 풀린 항목 로드** 버튼을 누릅니다.
6. 난동구리 프로젝트 안의 `browser-extension` 폴더를 선택합니다.
7. **폴더 선택** 버튼을 누릅니다.
8. **난동구리 브라우저 감지기**가 목록에 나타나고 켜져 있는지 확인합니다.
9. ChatGPT와 Gemini 탭을 `Ctrl + R`로 새로고침합니다.
10. 난동구리 설정 패널에서 상태가 **준비됨**인지 확인합니다.

## 설치가 잘됐는지 시험하기

1. 난동구리 앱을 실행해 둡니다.
2. 메뉴바 또는 시스템 트레이의 난동구리 아이콘을 왼쪽 클릭합니다.
3. 상태가 **준비됨**인지 확인합니다.
4. ChatGPT 또는 Gemini를 엽니다.
5. 답변이 조금 길게 나오는 질문을 입력합니다.
6. 답변이 만들어지는 동안 다른 탭이나 다른 앱으로 이동합니다.
7. 답변이 끝나면 작은 너구리가 화면에 나타나는지 확인합니다.
8. 너구리를 클릭합니다.
9. 질문했던 ChatGPT 또는 Gemini 페이지로 이동하면 정상입니다.

## 자주 막히는 부분

### `manifest.json` 파일을 선택했는데 설치되지 않아요

파일을 선택하면 안 됩니다. `manifest.json`이 들어 있는 **`browser-extension` 폴더 전체**를 선택해야 합니다.

### 설치 버튼이 보이지 않아요

확장 프로그램 화면의 **개발자 모드**를 먼저 켜야 설치 버튼이 나타납니다.

### 설치했는데 설정 패널이 `연결 확인 중`이에요

다음 순서로 확인합니다.

1. 난동구리 데스크톱 앱이 실행 중인지 확인합니다.
2. 확장 프로그램 카드의 스위치가 켜져 있는지 확인합니다.
3. ChatGPT 또는 Gemini 탭을 새로고침합니다.
4. 3~5초 기다린 뒤 설정 패널을 다시 확인합니다.
5. 그래도 안 되면 확장 프로그램 카드의 새로고침 아이콘을 누른 뒤 AI 탭도 다시 새로고침합니다.

### Chrome을 껐다 켜면 다시 설치해야 하나요?

아니요. 한 번 설치하면 그대로 유지됩니다. 확장 프로그램 폴더를 다른 위치로 옮기거나 삭제하지 않는 것이 좋습니다.

### 확장 프로그램을 지우고 싶어요

`chrome://extensions` 또는 `edge://extensions`에서 **난동구리 브라우저 감지기** 카드의 **삭제** 버튼을 누릅니다.
